## ğŸ—ï¸ Technical Architecture

### Why This Stack?

**PaddleOCR-VL** handles the vision layerâ€”it's specifically trained on degraded documents and handles the chaos of water stains, foxing, and ink bleed better than alternatives I tested. It gives us text extraction *plus* document quality analysis in one pass.

**ERNIE 4.5 (21B-A3B)** powers the four language agents. Each has a different system prompt, different expertise, different personality. They collaborate naturallyâ€”building on each other's findings, cross-validating results, and working together toward the goal. The key insight: having agents *collaborate in parallel* produces better results than any single model working alone, and it's 40-50% faster than sequential processing.

**WhatsApp-Style Interface** makes AI collaboration feel natural and familiar. Everyone knows WhatsAppâ€”2+ billion users worldwide. By using this familiar chat interface, we eliminate the learning curve and build instant trust. Users see agents working together like a professional team discussion.

**Server-Sent Events (SSE)** streams every agent's thinking to the frontend in real-time. This isn't just cosmeticâ€”watching the process unfold helps users spot issues early and builds trust in the output.
